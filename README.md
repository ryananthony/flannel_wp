flannel_wp
==========

Wordpress theme for my 90's cover band Flannel. Built on the [_s starter theme](http://underscores.me/).

![Screenshot of Flannel Theme](https://raw.github.com/ryananthony/flannel_wp/master/screenshot.png)

----------------
